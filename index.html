Cómo configurar Apps Script:
1. Abre script.google.com → pega el Code.gs
2. Ejecutar → initSheets (crea las hojas)
3. Implementar → Aplicación web
4. Ejecutar como: Yo (tu cuenta Google)
5. Acceso: Cualquier usuario, incluso anónimo
6. Copia la URL → pégala arriba → 💾 Guardar
7. En PC: ⬆️ Subir a nube para enviar datos
8. En celular: escanea el QR o pega la URL → ⬇️ Actualizar
